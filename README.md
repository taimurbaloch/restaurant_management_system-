# 🍽️ Restaurant Management System

A modern and responsive Restaurant Management System developed using PHP, MySQL, Bootstrap 5, HTML5, CSS3, and JavaScript. This project provides an easy-to-use admin panel for managing restaurant operations including reservations, menu items, customer accounts, messages, and orders.

---

## 📌 Features

### 👨‍💼 Admin Panel
- Secure Admin Login
- Interactive Dashboard
- Live Statistics
  - Total Orders
  - Total Customers
  - Confirmed Reservations
  - Messages Count
- Manage Reservations
- Manage Customers
- Manage Contact Messages
- Profile Settings
- Logout System

### 👥 Customer Side
- User Registration
- User Login
- Browse Restaurant Menu
- View Food Gallery
- Book Table Reservation
- Contact Restaurant
- Responsive User Interface

```text
config/db.php
```

---

## ⚙️ Installation

1. Download or Clone the Repository

```bash
git clone https://github.com/yourusername/Restaurant_Management.git
```

2. Move the project to:

```text
xampp/htdocs/
```

3. Start

- Apache
- MySQL

from XAMPP Control Panel.

4. Import the database into phpMyAdmin.

5. Open your browser:

```text
http://localhost/Restaurant_Management/
```
